# Audit Pipeline Data Files

This directory contains **sensitive and large data files** required for the audit pipeline. These files are **intentionally excluded from version control** via `.gitignore`.

## Required Files

| File | Size | Description | Source |
|------|------|-------------|--------|
| `dam_assets.json` | ~250 MB | Complete Aprimo DAM asset catalog | Export from Aprimo DAM API |
| `citizensbank_urls.txt` | ~500 KB | List of URLs to crawl for image discovery | Web crawl or sitemap export |
| `.audit_secret` | 64 bytes | HMAC key for secure asset fingerprinting | Generated via `generate_audit_secret.py` |

## Generated Files (Checkpoints)

The audit scripts create intermediate checkpoint files in this directory:

- `citizens_images_index.json` - Discovered images from URL crawl
- `dam_fingerprints.json` - Perceptual hashes of DAM assets
- `citizens_fingerprints.json` - Perceptual hashes of crawled images
- `asset_matches.json` - Matched assets between DAM and website
- `preflight_report_*.txt` - Validation reports with timestamps

These files are also excluded from git and can be regenerated by running the audit pipeline.

## Setup on New Machine

1. **Obtain source data files** from secure storage (DAM admin, data team, or previous developer)
2. **Place files in this directory**:
   ```
   assets/audit/dam_assets.json
   assets/audit/citizensbank_urls.txt
   ```
3. **Generate audit secret**:
   ```powershell
   python scripts/generate_audit_secret.py
   ```
4. **Run preflight check**:
   ```powershell
   python scripts/preflight_check.py
   ```

## Security Notes

⚠️ **DO NOT COMMIT THESE FILES TO GIT**

- `dam_assets.json` contains proprietary asset metadata
- `citizensbank_urls.txt` may contain internal/unpublished URLs
- `.audit_secret` protects asset fingerprint integrity

These patterns are protected in `.gitignore`:
```
assets/audit/*.json
assets/audit/*.txt
assets/audit/.audit_secret
```

## Troubleshooting

**Missing dam_assets.json:**
- Contact DAM administrator for latest export
- Verify file size matches expected ~250 MB
- Check JSON validity: `python -m json.tool assets/audit/dam_assets.json > nul`

**Missing citizensbank_urls.txt:**
- Run `scripts/01_crawl_citizens_images.py` to regenerate from sitemap
- Or obtain from previous audit run

**Missing .audit_secret:**
- Run `python scripts/generate_audit_secret.py`
- This generates a new random 64-byte hex key
- Note: New key invalidates previous fingerprints (you'll need to regenerate them)

## File Validation

The `setup_new_machine.ps1` script validates these files exist and meet size requirements:

- `dam_assets.json` must be ≥100 MB
- `citizensbank_urls.txt` must contain ≥100 URLs
- `.audit_secret` must be exactly 64 hex characters

Run the setup script to verify your environment:
```powershell
.\setup_new_machine.ps1
```
